<h1>About</h1>
libnifalcon is an open source, cross platform driver for the "Novint Falcon Haptic Device":http://home.novint.com. It provides communications and a kinematics model for the controller for all major operating systems. 

Please note that libnifalcon only provides control functions for the falcon, and is not a haptics scene graph, like "H3D":http://www.h3d.org or "Chai3D":http://www.chai3d.org.
